# vim-settings

## setup instructions

- open terminal (in Windows, use Git Bash)

```
cd $HOME
git clone https://github.com/tobiasrick/vim-settings.git
cd vim-settings
./win-setup.sh
```

- open gvim (on Windows: add to path if necessary)

- within gvim, install plugins:

```
<ESC>:PluginInstall
```
